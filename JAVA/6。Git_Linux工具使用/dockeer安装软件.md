##  分布式和集群是什么 

分布式是将服务放在不同的设备上运行，采取远程调用服务，让使用人员感觉 整个应用 像是在一个服务器上，之所以要分布，也是减少服务的ou合性， 不会像一体的，出现一个服务出现问题影响项目 ，另外的项目就无法使用，

集群是将一个项目 放在很多 台设备上运行，心减缓压力





## Docker 

docker   ，是基于镜像的

在Linux系统中，我们安装redis mysql  要下载配置很多文件，还要弄什么环境变量什么的， 稍一弄错，会影响 其他软件或者系统的运行，

这时，docker 便出现了，它是一个可以装载镜像的软件   可以 理解为它有一键ghost的功能 

### 镜像

镜像就是别人设置好的软件或者系统，只要我们去下载 就可以直接来用的一种文件 

而docker是可以装载这些镜像。docker  直接去下载所需要软件的镜像，然后将软件放到一个容器， 这个容器就是一个小的Linux系统，每个容器相互不影响
