# js对象，json字符串 的区别 

js对象 是通过 大括号，｛"name" =："key "， "  value" ："张三"，......｝的形式来存在的

双引号可改为单绰号，   这就是js对象 ， json字符串就是js对象 的一种，



json字符串  严格 按照双引号来写，j k  v 都要用双引号引着才是json字符串 ，
